# MindMark
MindMark is a native macOS and iOS knowledge management app built with SwiftUI. Notes are stored as plain Markdown files in a folder you choose — no proprietary formats, no forced sync. Bring your own cloud (iCloud Drive, Dropbox, etc.).

The editor is powered by BlockNote (React/TypeScript) embedded via WKWebView, giving you a modern block-based writing experience while keeping your data fully portable. A companion .json file preserves rich formatting like font colors and block metadata without touching the human-readable .md file.

Built-in AI tools (summarize, expand, simplify, translate, and more) connect directly to the Anthropic Claude API using your own API key.
